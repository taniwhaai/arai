# Arai

Rust CLI that enforces AI coding assistant instruction files (CLAUDE.md, .cursorrules, etc.) via Claude Code hooks. Single binary, local-first, ~32ms median match cost.

## Architecture (existing)

See `CLAUDE.md` at repo root. Key modules: `parser.rs` (7-layer rule extraction), `hooks.rs` (PreToolUse/PostToolUse), `guardrails.rs` (matching), `intent.rs` (severity classification), `audit.rs` (local JSONL log), `compliance.rs` (Pre/Post correlation).
